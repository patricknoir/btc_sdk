Buffer

Buffer class Null safety

This class is used to perform manipulation of array of bytes represented as Uint8List.

Constructors

Buffer({Uint8List? initial})
Create a new buffer instance. Optionally an initial value can be specified, otherwise it will create an empty buffer.

inspectSegment(int end, {int start = 0, Endian endian = Endian.little}) → Uint8List
Export a segment from the buffer as a Uint8List, if the segment is requested as Endian.big the list will be reversed.
inspectUint16({Endian endian = Endian.little}) → int
Expose the first 2 bytes of the buffer as an integer value. Reverse the 2 bytes if the Endian specificed is Endian.big. The buffer will suffer no changes with this operation. It throws an error if the Buffer size is smaller than 2 bytes.
inspectUint32({Endian endian = Endian.little}) → int
Expose the first 4 bytes of the buffer as an integer value. Reverse the 4 bytes if the Endian specificed is Endian.big. The buffer will suffer no changes with this operation. It throws an error if the Buffer size is smaller than 4 bytes.
inspectUint64({Endian endian = Endian.little}) → int
Expose the first 8 bytes of the buffer as an integer value. Reverse the 8 bytes if the Endian specificed is Endian.big. The buffer will suffer no changes with this operation. It throws an error if the Buffer size is smaller than 8 bytes
inspectUint8() → int
Expose the first byte from the Buffer without modifying the buffer content.
inspectVarInt({int start = 0}) → VarInt
Parse end return the VarInt field from the specificed start position from this buffer. This operation doesn't modify the buffer content.

pullSegment(int end, {int start = 0, Endian endian = Endian.little}) → Uint8List
Extract a segment from the Buffer. If endian is Endian.big the segment will be reverted. The buffer is modified with the extracted segment removed.
pullUint16({Endian endian = Endian.little}) → int
Extract the first 2 bytes of the buffer as an integer value. Reverse the 2 bytes if the Endian specificed is Endian.big. The buffer will be modified after this operation, and the first 2 bytes will be removed. It throws an error if the Buffer size is smaller than 2 bytes.
pullUint32({Endian endian = Endian.little}) → int
Extract the first 4 bytes of the buffer as an integer value. Reverse the 4 bytes if the Endian specificed is Endian.big. The buffer will be modified after this operation, and the first 4 bytes will be removed. It throws an error if the Buffer size is smaller than 4 bytes.
pullUint64({Endian endian = Endian.little}) → int
Extract the first 8 bytes of the buffer as an integer value. Reverse the 8 bytes if the Endian specificed is Endian.big. The buffer will be modified after this operation, and the first 8 bytes will be removed. It throws an error if the Buffer size is smaller than 8 bytes.
pullUint8() → int
Extract the first byte from the buffer, after this operation the buffer content will be modified and the pulled element will be removed from the head. It throws an error if the Buffer is empty.
pullVarInt({int start = 0}) → VarInt
Extract a parsed VarInt from the buffer at the start position specified. Once the VarInt is returned the field is also removed from the buffer.
